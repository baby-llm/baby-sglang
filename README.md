# baby-sgl

Minimal, hackable LLM inference playground inspired by SGLang. Clarity first, performance second.

## Quick Start

```bash
python run_demo.py --comprehensive --model-id Qwen/Qwen2.5-0.5B --max-new-tokens 128 --seed 42 --device auto
```
